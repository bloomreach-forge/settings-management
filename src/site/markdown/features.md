## Features

The Settings management plugin covers the following features:

+ Maintenance
    + Form data
    + Event log
+ Uploads
    + Configure image upload settings
        + Simultaneous threads
        + Max image size
        + etc
    + Configure asset upload settings
        + Simultaneous threads
        + Max file size
        + etc
+ Security
    + Login page security
    + Password policies
+ Other
    + Social sharing configuration
